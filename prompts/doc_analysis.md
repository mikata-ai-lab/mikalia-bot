# Prompt: Análisis de Documento

Analiza el siguiente documento y extrae los puntos más relevantes para escribir un blog post.

## Documento: {doc_name}
## Formato: {doc_format}

## Contenido:
{content}

## Instrucciones:
1. Identifica el tema principal del documento
2. Extrae los puntos clave y argumentos principales
3. Encuentra datos, estadísticas, o ejemplos interesantes
4. Identifica qué sería más valioso para los lectores del blog
5. Resume en formato bullet points los aspectos más relevantes

## Formato de respuesta:
Responde con un resumen estructurado que capture la esencia del documento
y pueda usarse como base para un blog post educativo.
